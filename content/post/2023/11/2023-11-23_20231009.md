<p>西田尚輝. (2023). 「How の問いから Why の問いへ: 歴史社会学における因果と理論」『社会学評論』<em>74</em>(1), 17-33. <a href="https://cir.nii.ac.jp/crid/1520860078908497920">https://cir.nii.ac.jp/crid/1520860078908497920</a>.<br />※他のメモから移植。</p>
</blockquote>
<ul>
<li><strong>"歴史社会学者が単一事例研究から外的妥当性のある理論を作り出すためにはどうしたらよいのか？"（＝一般化可能な理論の洗練）</strong>
<ul>
<li>⇒<strong>「さまざまな説明変数が各時点の歴史的アクターの可能な選択肢としてどのように出現し, どのように議論され, 最終的に断念されたかを綿密に追跡」</strong></li>
</ul>
</li>
<li>差異法（Mill; Skocpol 1979; Skocpol &amp; Somers 1980）, 量的研究（KKV）
<ul>
<li>⇔エスノグラフィー: 「抽象的な変数の同定ではなく, 変数それ自体の特徴や性質がどのように獲得され, 因果効果をもつにいたったのかを具体的に特定」（23）</li>
</ul>
</li>
<li>社会的構築主義 in IR
<ul>
<li>Wendt (1998): 「構成理論 constitutive theory」</li>
<li>そもそもどのようにXやYが構成されているか（Howアプローチ）
<ul>
<li>因果にかかわらないものとして規定</li>
</ul>
</li>
</ul>
</li>
<li>⇔構成的主張 in エスノグラフィー
<ul>
<li>「因果的説明とは別のものだが, ある現象や特性がどのように存在するようになったかを丹念に記述することによって, 因果的説明を発展させうる」</li>
</ul>
</li>
<li>⇒形成物語（Hirschman &amp; Reed 2014）：新しい社会的類型（アクター, 組織形成, 概念など）の歴史的形成を追跡
<ul>
<li>e.g. 「〇〇とは何か」「〇〇の誕生」「〇〇の発明」</li>
<li>「形成物語は, 量的研究の抽象的な変数を拒否し, かわりに, 社会現象がどのように生まれ, 再形成され, 量的研究者が注目するような因果効果をもつようになったかを追跡することによって, 因果的説明を展開しようとするもの」（26）</li>
</ul>
</li>
<li>負の事例分析
<ul>
<li>e.g. Emigh (1997); Moore (1966=2019); Skocpol (1979)</li>
<li>新たな変数の発見 +理論そのものの再構築・精緻化する機会の提供<br />→他方で外的妥当性に弱さ</li>
</ul>
</li>
<li>単一事例研究における理論構築
<ul>
<li>当時のアクターにとって選択可能であったにもかかわらず、最終的に断念された選択肢に注目
<ul>
<li>cf. 「可能性原則 possibility principle」（Mahoney &amp; Goerts 2004）</li>
<li>歴史的アクターが実質的に認識していた説明変数<br />e.g. 可能性はあったが諦めた選択肢；試したが失敗した選択肢
<ul>
<li>⇒その決定が引き起こした結果に関する議論を精緻化</li>
</ul>
</li>
</ul>
</li>
<li>Ermakoff (2019: 593-5)
<ul>
<li>a) 生成過程に関与するアクター集団を明確に定義、b) 行動と相互作用を時間的に展開するとおりに記録、c) 前向きの結果（forward-looking outcome）を採用
<ul>
<li><strong>comment: 「前向きの結果を採用」とは？</strong></li>
</ul>
</li>
<li>結果から遡及すれば後知恵バイアスを誘発するおそれ</li>
</ul>
</li>
<li>インタビュー（オーラルヒストリー）；一次資料（歴史的アクターが作成した資料）
<ul>
<li>日付入り行政文書、アクターによって収集された官報・新聞・雑誌の切り抜き、アクターの日記・手紙・回顧録 etc.</li>
</ul>
</li>
</ul>
</li>
