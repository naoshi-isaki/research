<blockquote>
<p>Krcmaric, D., Nelson, S., &amp; Roberts, A. (2023). Billionaire Politicians: A Global Perspective. <em>Perspectives on Politics</em>, 1-15. [https://doi.org/10.1017/S1537592723002761].</p>
</blockquote>
<ul>
<li>研究の目的：「世界中の政治に超富裕層が直接関与していることについては、体系的な証拠が不足」→公式的な政治参加に関する体系的な証拠を提示</li>
<li>分析手法, データセット（資料）<br />
<ul>
<li>独自のデータセット: Billionaires in Politics Around the World (BIPAW) dataset</li>
<li>the Forbes Billionaires List (N=2,072)</li>
</ul>
</li>
<li>結果<br />
<ul>
<li>
<blockquote>
<p>億万長者の政治家は驚くほど一般的な現象であり、世界の億万長者の11％以上が、公式的な政治職に就いているか、政治職に就こうとしている。</p>
</blockquote>
<ul>
<li>11.7%（242/2,072）→非公式に限るとより大きい。</li>
</ul>
</li>
<li>当選しやすい理由
<ol>
<li>莫大な個人資産を高額な運動資金に充当　e.g. バビシュ</li>
<li>メディア企業を所有　e.g. ベルルスコーニ</li>
<li>非拘束性（いつ、どこに出馬するか）、機会主義的に行動可能</li>
</ol>
</li>
<li>億万長者の政治参入率は、民主主義国家よりも独裁国家の方が高い
<ul>
<li>コネクションの形成；支配体制への忠誠心の提示</li>
</ul>
</li>
<li>億万長者の政党のほぼ4分の3がイデオロギー上で中央値より右側に位置</li>
</ul>
</li>
</ul>
<ul>
<li>コメント等<br />
<ul>
<li>間接的影響力ではなく、公式的な政治参加に着目。確かにニッチだった。</li>
<li>民主主義国家における参入動機は何なのか？というのは本論文の射程外だろうが、気になるところ。</li>
</ul>
</li>
</ul>
