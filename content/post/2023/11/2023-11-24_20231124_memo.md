<h5>第6章　育児休業制度の効果はどこにみえるのか――働き方、賃金と夫婦の家事・育児分担の変化</h5>
<ul>
<li>内容抜粋
<ul>
<li>「出産後の妻は、余暇時間を犠牲にしながら仕事と家事・育児の両立を行っている可能性が危惧される」（167）（図6-5）</li>
</ul>
</li>
<li>コメント等
<ul>
<li>「性別役割分業」「3歳児神話」賛成を変数として分析しているのは面白い（表6-1）
<ul>
<li>いずれも賛成しない人のほうが実際に育児休業を取得して就業継続する傾向（156-7）</li>
<li>育児休業資格があったにもかかわらずそれを利用しなかった人は、前者の意識は強くない一方で、後者を支持している割合は高い（158）</li>
</ul>
</li>
<li>もちろん第3節5項のような偏りがありながらも、夫婦（パートナー）間で納得がついているならば問題ないのだが。</li>
<li>男性の意識の問題で片づけるべきでない、という点は同意。
<ul>
<li>
<blockquote>
<p>家事・育児負担が減らないままに働くことを選択する女性が増えることは、女性の厚生を下げ、長期的には女性活躍を阻害したり少子化を加速させたりする結果をもたらしかねない。<span style="text-decoration: underline;">子どもの誕生後も男性の育児時間が大きく増えないのは、父親としての自覚がないといった当事者の意識問題だけで片づけるべきではなく、乳幼児がいても男性の労働時間がほとんど変わらないことや、残業手当を含む収入の減少を嫌うことなどにより、女性に比べて育児にまつわる制度の利用が男性では著しく低い現状をどのように打破していくかが今後は重要になっていくだろう。</span>育児休業制度は女性労働者のためだけにあるべきではなく、男女双方が制度を利用しながら、ともに子どもを育てることのできる社会を実現させることが求められている。（170）</p>
</blockquote>
</li>
</ul>
</li>
</ul>
</li>
</ul>
