<ul>
<li>目的
<ul>
<li>政治におけるポピュリズム的レトリックの役割</li>
<li>ポピュリズムが特定の政策ポジションと相性が良いのか、あるいはその逆なのか；実質的な政策ポジションとポピュリズムの間に相互作用効果はあるのか</li>
<li>
<p><a href="https://doi.org/10.1017/XPS.2022.9" target="_blank">Neuner and Wratil 2022</a>; <a href="https://doi.org/10.1017/XPS.2022.9" target="_blank">Castanho Silva et al. 2022</a></p>
</li>
</ul>
</li>
<li>方法
<ul>
<li>コンジョイント実験<br />→回答者別のクラスター化ロバスト標準誤差を用いた単純線形回帰、候補者選択における様々な属性の平均限界成分効果（AMCE）を推定
<ul>
<li>バイデン政権下の米国下院予備選挙における現実的な選挙メッセージに、ポピュリズム的・非ポピュリズム的レトリック、左右の経済政策、開放的・閉鎖的な移民政策の立場を埋め込み</li>
<li>回答者は、ポピュリスト的なレトリックと政策立場の組み合わせが異なる、ランダムに生成された候補者のキャンペーン・メッセージの組から、好みの候補者を選択</li>
</ul>
</li>
<li>アメリカ, 2022年4月, N=1004（共和党員36％、民主党員58％）</li>
</ul>
</li>
<li>結論
<ul>
<li>ポピュリズムが票を獲得する上で相対的に効果がなく、一致した政策優先事項の方が一貫して重要；実質的な政策立場の一致が投票選択にとって最も重要な要因<br />
<ul>
<li>前回のトランプ政権時の実験（<a href="https://doi.org/10.1017/XPS.2022.9" target="_blank">Castanho Silva et al. 2022</a>）と同様、現職がポピュリストか否かは関係ない</li>
</ul>
</li>
<li>
<p>ポピュリズム的レトリックと主要政策ポジションの間に有意な相互作用効果もなし</p>
<ul>
<li>
<blockquote>
<p>人々はしばしば移民排斥政策をポピュリズムと結びつけるが、ポピュリズムのフレームが移民排斥の立場を有権者にとってより魅力的なものにするわけではなく、その逆もまた然りである。</p>
</blockquote>
</li>
</ul>
</li>
<li> ポピュリスト政党や候補者の選挙的成功のかなりの部分は、ポピュリズム的アピールではなく、そのプログラム的政策ポジションにあることを示唆<br />
<ul>
<li>
<blockquote>
<p> 　しかし、だからといってポピュリズムが重要でないとか、まったく役に立たないというわけではない。有権者の投票選択にはあまり影響を与えないかもしれないが、ポピュリズムはメディアの注目を集め、世間への露出を増やすという点では効果的かもしれない。例えば、ポピュリズム的なコミュニケーションは、ソーシャルメディア上でより多くの政治的関与を生み出すことが分かっている。また、有権者を投票に動員する上でも効果的かもしれない。<br />　ポピュリズムが票を獲得する能力を過大評価し、他の有意義な政治的・社会的舞台でポピュリズムが果たす役割を過小評価することに注意すべきである。しかし、ポピュリズムの影響と結果を正しく認識するためには、ポピュリズムとそのホストのイデオロギーを切り離すことが極めて重要である。</p>
</blockquote>
</li>
</ul>
</li>
</ul>
</li>
