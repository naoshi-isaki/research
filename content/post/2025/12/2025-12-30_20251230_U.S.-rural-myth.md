<ul>
<li>神話1：人口減少によりアメリカの郊外は消滅しつつある<br />
<ul>
<li>多くの人々はアメリカの農村部から人口が流出していると考えている。しかし実態はもっと複雑だ。確かに2010年から2020年にかけてほとんどの農村郡で人口が減少した。しかし約3分の1は増加した。</li>
<li>
<p>郊外人口の変化について知っておくべき重要な点は、「郊外」と定義される地域が時間とともに変化するということだ。ある農村町が十分に成長すると、米国行政管理予算局はそれを「都市」に再分類する。つまり、アメリカの農村が消えつつあるわけではない——変化し、時には都市化しているのだ。</p>
</li>
</ul>
</li>
<li>神話2：ほとんどの郊外のアメリカ人は農場に住んでいる<br />
<ul>
<li>
<p>現在、農村部の雇用で最も大きな割合を占めるのはサービス業の職種であり、小売業、飲食業、在宅医療、宿泊・飲食サービスなどが該当する。こうした職種は賃金が低く、福利厚生が乏しく、勤務時間が不安定な場合が多く、多くの農村世帯が経済的な安定を維持することを困難にしている。</p>
</li>
</ul>
</li>
<li>神話3：アメリカの郊外には白人しか住んでいない
<ul>
<li>
<p>人々はしばしばアメリカの農村部を主に白人だと想像するが、それは全体像ではない。農村住民の約4人に1人は非白人である。ヒスパニック系と黒人が最大の割合を占め、先住民は他の人種グループよりも農村地域に居住する人口比率が高い。</p>
</li>
</ul>
</li>
<li>神話4：アメリカの郊外は都市部より健康である<br />
<ul>
<li>
<p>多くの人々は田舎の生活が都会より健康的だと想像する。しかし真実は逆だ。地方住民は都市住民より若くして亡くなり、死亡率も高い。学者はこれを「地方の死亡率ペナルティ」と呼び、この差は長年拡大を続けている。COVID-19パンデミックにより、地方コミュニティでの死亡率上昇が原因でこの格差はさらに拡大した。</p>
</li>
</ul>
</li>
<li>神話5：郊外の家族は都市部より伝統的である<br />
<ul>
<li>
<p>今日、農村部の子供たちは都市部の子供たちよりも結婚した両親と同居している可能性が低く、未婚の同棲カップルと暮らしたり、祖父母やその他の親族の世話を受けている可能性が高い。この結果の一部として、農村部の児童貧困率は都市部より高い。多くの農村世帯はSNAP（食品支援プログラム）などのセーフティネット支援に依存している。農村世帯は多様であり、多くが経済的に脆弱な状態にある。</p>
</li>
</ul>
</li>
<li>神話 6：新たな「郊外の反乱」がドナルド・トランプに大統領選挙での勝利をもたらした
<ul>
<li>
<p>1980年代から1990年代にかけて、農村部と都市部の有権者間の党派格差は平均3ポイントであったが、2000年代には10ポイント、近年の選挙サイクルでは20ポイントに拡大した。<span style="text-decoration: underline;">したがって、トランプ氏の農村部支持は新たな「反乱」ではなく、長期的な傾向の一部であった。</span></p>
<p>そして2024年、地理的に見れば重要な動きは農村部の有権者ではなかった――大都市圏での投票率の急落だった。両候補とも2020年より都市部の得票数を減らし、カマラ・ハリスは主要都市と中規模都市でジョー・バイデンが4年前に獲得した票より1000万票以上少ない票しか得られなかった。</p>
</li>
</ul>
</li>
</ul>
