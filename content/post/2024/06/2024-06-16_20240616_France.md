<ul>
<li>
<blockquote>
<p>かつて政治の片隅に追いやられていたナショナリスト、ポピュリストのRNを政府の中枢に送り込む準備はできているのだろうか？</p>
<p>その答えは、6月9日の欧州選挙でRNに惨敗したマクロンが、早期の議会選挙を呼びかけた衝撃的なキャンペーンが終わり、3週間余りで出るだろう。</p>
<p>6月30日の世論調査と7月7日の第2回投票の結果は、マリーヌ・ルペンが10年以上を費やして、よりスムーズでプロフェッショナルな勢力に作り上げた現在のRNが、有権者にその任務を果たせると映るかどうかに大きく左右される。</p>
<p>ダニエルが住むフランス中部のベリー地方での投票パターンは、RNがいかに新しい地域や有権者層に進出しているかを示している。過疎化、高速鉄道の少なさ、経済の弱さなどから、学者たちが「空虚の斜線 the diagonal of emptiness」と呼ぶフランスの一帯を征服しつつあるのだ。</p>
</blockquote>
</li>
<li>
<blockquote>
<p>数十年にわたってフランスの極右を研究してきた学者で作家のパスカル・ペリノーは、有権者がRNが政権を主導することを望んでいるという考えをもはや否定できないと言う。</p>
<p>「ほんの数年前なら、彼らの勝利はあり得ないと言っていただろう。今となっては、可能性はあるし、可能性さえある」。</p>
</blockquote>
</li>
</ul>

<ul>
<li>
<blockquote>
<p>歴史的に左派が多いサン＝ブノワ＝デュ＝ソーでも、ルペンの支持率は過去10年間で着実に上昇している。今年、人口550人のフラワーボックスで埋め尽くされた絵のように美しい中世の村では、最後のブーランジェリーと肉屋が閉店した。2019年に地元の調理器具工場が閉鎖されたことも痛手だった。</p>
<p>ルペンは、パリや裕福なマクロン支持者の人口が持つ富、権力、文化的価値から遠く離れた、彼女が「忘れられたフランス」と呼ぶそのような場所の擁護者として自らを位置づけている。</p>
<p>サン＝ブノワ＝デュ＝ソーの左派系若手市長ダミアン・バレは、ビジネスやサービスを維持するために戦うことで、人々の衰退感と闘う決意を固めている。彼のプロジェクトには、造園、建物の修復、文化プログラムなどがある。彼は昨年、村の高齢化したヤギの群れと交尾する雄ヤギを探す旅にも出た。</p>
<p>取り残されているという感覚はRN票の強力な原動力ではあるが、その魅力は今やそれ以上だと彼は言う。</p>
<p>「RNへの投票は抗議票であり、うんざりした票であると10年来言われてきたが、マリーヌ・ルペンが党を牛耳ってからは支持票なのだ。人々は、『私は彼らのプログラムを支持している』と言うだろう」。</p>
</blockquote>
</li>
<li>
<blockquote>
<p>全国レベルでは、RN投票の社会学も変化している。</p>
<p>イプソスの分析によれば、欧州選挙では60代の34％がRNに投票し、2019年の23％から上昇した。彼らは、長年大統領に忠誠を誓い、有権者の3分の1を占める年金受給者の間では、マクロン大統領の同盟を上回ってさえいた。</p>
<p>イプソスの世論調査担当者、ブリス・タイントゥリエは言う。「彼らは万能政党になったのです」。</p>
</blockquote>
</li>
</ul>

<p><iframe src="https://hatenablog-parts.com/embed?url=https%3A%2F%2Fwww.ft.com%2Fcontent%2F32418e1e-4377-451d-a52e-89e2b5e14fa8" title="How does ‘cohabitation’ work in French politics?" class="embed-card embed-webcard" scrolling="no" frameborder="0" style="display: block; width: 100%; height: 155px; max-width: 500px; margin: 10px 0px;" loading="lazy"></iframe><cite class="hatena-citation"><a href="https://www.ft.com/content/32418e1e-4377-451d-a52e-89e2b5e14fa8">www.ft.com</a></cite></p>
<ul>
<li>
<blockquote>
<p><strong>コアビタシオンは一般的か？</strong><br />フランソワ・ミッテラン大統領の社会党が議会選挙でジャック・シラク率いる中道右派に敗れ、シラクが首相に就任した1986年から88年、ミッテラン党が再び右派に敗れ、エドゥアール・バラデュールを首相に任命した1993年から1995年、そして当時大統領だったシラクが解散総選挙に打って出た1997年から2002年である。</p>
<p>それ以来、このような例はない。大統領の任期は7年から5年に短縮され、議会の任期と同じになった。また、選挙日程が同期化されたため、大統領選の直後に立法府の投票が行われ、しばしば結果が確定するようになった。マクロンの世論調査は、この状況を一変させるかもしれない。</p>
</blockquote>
</li>
<li>
<blockquote>
<p><strong>誰が首相を選ぶのか？</strong><br />厳密には、野党の議席数に関係なく大統領が首相を選ぶ。しかし、大統領が選ぶ首相と彼らが樹立する政府は、国民議会の支持を得なければならない。</p>
<p>大統領の政党が絶対過半数または相対多数を確保した場合、大統領は首相指名権を保持する。野党グループ（この場合、おそらくジョルダン・バルデラを首相候補とするRN）が絶対多数を得た場合、そのグループは自らの選択を押し通す力を持つ。</p>
<p>より可能性の高いシナリオとして、絶対多数が存在しない場合、大統領は人事に影響を与えることができるが、政権を組み立てるために他のプレーヤーと交渉する必要がある。専門家は、フランス憲法には政権樹立の具体的な方式が定められていないため、大統領に操縦の余地があると指摘している。</p>
<p>しかし、そのような政権が存続する保証はなく、マクロンは再び、明確な多数派を持たない政権運営を試みることになるかもしれない。</p>
</blockquote>
</li>
<li>
<blockquote>
<p><strong>首相と政府にはどれほどの権力があるのか？</strong><br />認識とは裏腹に、非常に多い。マクロンは、非常に行政的でゴーリスト的な大統領を率い、政府を厳しく統制している。しかし憲法は、政府の行動を指揮し、法律の執行を確保するのは首相であると明記している。大統領は最高司令官であり、核のボタンをコントロールし、外交政策を支配しているが、政府は国防の一部にも責任を負っている。</p>
<p>政権が過半数を占めると仮定すれば、そのアジェンダを実行に移す余地はかなりあるだろう。</p>
<p>シラク首相は、わずかな過半数しか得られなかったにもかかわらず、企業の国有化や半比例制の導入など、左派の前任者たちが実施した多くの改革を覆すことができた。同じく右派のバラデュールは、年金の保険料納付期間を40年に引き上げ、民営化を継続した。左派のジョスパン政権は、最大週35時間労働制の導入に舵を切った。</p>
<p>RN政権が単独過半数を獲得すれば、理論上、移民を厳しく制限し、エネルギー政策の主導権をブリュッセルから奪い返し、公共部門の仕事や社会住宅で非国民よりもフランス人を優遇する「国民優先権」を確立するなどの計画を実行に移すことができる。</p>
<p>しかし、もしRNが単に最大政党であっただけで、完全な多数派ではなかったとしたら、その首相は不信任票を投じられやすくなる。</p>
</blockquote>
</li>
<li>
<blockquote>
<p><strong>大統領は政府を拘束できるのか？</strong><br />ある程度は。大統領の主な拘束力は選挙や国民投票を招集することだが、フランスの選挙規則によれば、大統領（全員男性）は12カ月に1度しか選挙を招集できない。国家元首は政令への署名を拒否することもできるが、行政府はそれを回避するために、政令を通常の法律とし、憲法第49条第3項、いわゆる、政府が自らを信任投票の対象とすることを利用して、政令を迅速に制定することができる。皮肉なことに、これは2022年以来、マクロンとその首相が、不従順な国民議会を克服するために頻繁に使ってきた手続きである。</p>
<p>大統領は外交と安全保障も掌握しているため、たとえばRN政権がウクライナに対するフランスの立場を根本的に変えたり、キーウに対するフランスの支援を縮小したりするのを阻止することができる。</p>
<p>最後に、大統領はテレビで国民に演説する権利など、公の場で政権を批判することができる。ミッテランは、事実上の野党指導者として、その演説の場を頻繁に利用した。</p>
</blockquote>
</li>
<li>
<blockquote>
<p>コアビタシオンは成功するか？<br />行政権の分割は、指導者が牽制と均衡によって制約されることを望む有権者にとって魅力的である。必ずしも政治的麻痺を招くわけではないが、マクロンが国民党政権を促進するために何かをするとは考えにくい。実際、マクロン大統領の意図は、フランスの有権者が2027年の任期満了時にルペン氏を大統領に選ばないように、国民党には能力がないことを示すことかもしれない。</p>
<p>マクロン大統領の盟友の一人は、もしRNが絶対多数を獲得した場合、マクロン大統領が辞任しない限り、政権樹立を拒否する可能性を示唆した。そうなれば、フランスは同居ではなく、政治的・制度的危機へと向かうことになる。</p>
</blockquote>
</li>
</ul>

<ul>
<li>
<blockquote>
<p>贅沢な公約を実現するための増税を挙げることもできる。あるいは借り過ぎによる金利上昇もある。しかし、移民を管理しなければならないことほど、ポピュリストの大義を害することはないだろう。低賃金部門における外国人労働者に対する彼らのもっともらしく聞こえる代替案は、国内労働者への賃金を上げるというものだが、国民の物価に対する敏感さが試されることになる。たとえ有権者が社会的ケアや小売コストの上昇を嫌がらなかったとしても、そのトレードオフはついに明らかになるだろう。テストされることのないポピュリストのアイデアには、偽りの信頼性がある。それを変えられるのは、政権に就いた時だけだろう。 </p>
</blockquote>
</li>
<li>
<blockquote>
<p>一度でも政権に就けば、ポピュリストは政権を維持するために制度を曲解するかもしれないし、選挙で落選させるというメリットよりも有害なことをするかもしれない、という議論こそがより良い議論なのだ。(世界最大の単一市場からの離脱など）。それゆえ、ドイツの主流派は強硬右派に対して「防疫線」を張っている。</p>
<p>侮れない議論である。理想的な世界では、権力に近づくだけでポピュリストは有権者を失うだろう。マクロンはこの夏、フランスに国民連合の首相を考えさせたいようだ。しかし、RNの勝利に建設的な要素がまったくないと考えれば、彼は選挙を行う決断をしなかっただろう。ある時点で、有権者は自分の表明した願望の結果と向き合わなければならない。</p>
</blockquote>
</li>
<li>
<blockquote>
<p>有権者にコントロールされた量を与える安全な方法はないかもしれない。しかし、ポピュリストがテレビに出たり、ステージに立ったりすることはあっても、それほど大きな責任を負わないという現状は維持できない。マクロンの選挙は、ほとんど小説的な運命の人による、もうひとつのとんでもないギャンブルに仕立て上げられるだろう。実際、マクロンにとって最も賢明な選択だったのかもしれない。</p>
</blockquote>
</li>
</ul>
